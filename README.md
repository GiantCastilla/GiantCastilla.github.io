# PROYECTO WEB: CREACIÓN DE PLATAFORMA EMPRESARIAL RESPONSIVE - VIBE DROP

## 📝 Descripción del Proyecto
Este proyecto consiste en el diseño, maquetación y desarrollo de una plataforma web empresarial responsive para **Vibe Drop**, una tienda ficticia dedicada a la comercialización de calzado urbano y deportivo de edición limitada. El objetivo es automatizar la visualización del stock en tiempo real y profesionalizar la experiencia de usuario.

El desarrollo se ha realizado de forma modular, implementando las mejores prácticas de ingeniería de software mediante la separación estricta de la estructura (HTML5), los estilos visuales (CSS3) y la lógica funcional del lado del cliente (JavaScript).

## 🚀 Características Principales
* **Maquetación Semántica:** Estructura limpia utilizando etiquetas nativas de HTML5.
* **Diseño Responsive Pro:** Adaptabilidad total para formatos de escritorio, tablet y dispositivos móviles mediante Media Queries y sistemas de rejilla (CSS Grid y Flexbox).
* **Carrusel Infinito Automático:** Galería dinámica continua con efectos interactivos (`hover/scale`) al interactuar con los modelos destacados.
* **Sección de Checkout Modular:** Interfaz de compras optimizada con validaciones dinámicas en JavaScript para pedidos personales o regalos.
* **Control de Animaciones:** Efectos visuales fluidos sincronizados con el desplazamiento del usuario a través de la API moderna `IntersectionObserver`.

## 🛠️ Tecnologías Utilizadas
* HTML5 (Estructura semántica y tablas de disponibilidad de tallas)
* CSS3 (Diseño responsivo, animaciones dinámicas y Grid Layout)
* JavaScript Nativo (Manipulación del DOM, lógica de formularios y eventos interactivos)
